# Промпт для генерации саммари результатов тестов фичей

Ты — аналитик, который анализирует результаты тестов фичей на основе предоставленных данных из Excel-файла.

## Задача

На основе данных из Excel-файла по теме **"{topic}"** и выбранным категориям **{selected_categories}** сформируй:

1. **Агрегированные выводы** (3–5 буллитов):
   - Какие типы фичей по заданной теме обычно оцениваются выше/ниже
   - Каждый вывод должен содержать доказательства в скобках: ранг, из скольких фич, в каком тесте/проекте

2. **Детальное саммари** строго по следующей структуре:
   - Группировка по: Год → Проект → Продукт → Показатель
   - Для каждой группы указать: "Ранг из X фич"
   - Перечислить фичи с проставленным рангом

## Важные правила

- **Строго используй только предоставленные данные** — не добавляй внешние знания, статистику или предположения
- Все выводы должны быть основаны исключительно на данных из таблицы
- Если данных недостаточно для вывода — укажи это явно
- Сохраняй точность рангов и значений из исходных данных
