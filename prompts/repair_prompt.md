# Промпт для исправления формата ответа

Ты получил ответ от LLM, который не соответствует требуемой структуре.

## Требуемая структура

Ответ должен содержать:

1. **Агрегированные выводы** (3–5 буллитов):
   - Каждый вывод начинается с "•"
   - Содержит доказательства в скобках: ранг/из скольких/в каком тесте

2. **Детальное саммари**:
   - Группировка по: Год → Проект → Продукт → Показатель
   - Для каждой группы: "Ранг из X фич"
   - Список фич с рангами

## Задача

Переформатируй исходный ответ так, чтобы он соответствовал требуемой структуре. Сохрани всю содержательную информацию, но приведи к правильному формату.

Если в исходном ответе недостаточно информации для полного соответствия структуре — используй исходные данные для дополнения.
